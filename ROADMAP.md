# Roadmap

## ✅ Built

- Authentication (email/password sign up, sign in, sign out)
- Tab navigation (Home, Map, Add, Settings)
- Spot list with search and category filters
- Full map view with category-colored markers
- Add spot form with all fields
- Spot detail screen with photo gallery
- Edit and delete spots
- Geolocation (permission, current position, distance)
- Photo upload (camera + gallery → Supabase Storage)
- Offline cache for spots list
- Design system (tokens, reusable components)
- Loading / error / empty states
- Settings screen
- Row Level Security on all tables

## 🔮 Future (Optional)

- Dark mode theme toggle
- Sort spots by distance
- Favorites-only filter
- Local notifications (reminders)
- Export/share a spot
- Onboarding walkthrough screens
- Profile avatar upload
- Spot visit history / check-in
